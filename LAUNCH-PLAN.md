## How to Publish Each Day

1. Open `src/lib/published-pages.ts`
2. Move 20 slugs from `PUBLISH_QUEUE` array → into `PUBLISHED_PAGES` Set
3. Run `vercel --prod` from your terminal
4. Done — middleware automatically allows Google to crawl the new pages
