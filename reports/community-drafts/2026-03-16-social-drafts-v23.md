### Thread 1 — Data Availability Wars (Friday Mar 20, 10am ET)

```
The battle for Ethereum's data availability layer is getting serious 🔥

Celestia. EigenDA. Avail.

Here's where things stand in 2026 🧵

1/ What IS a data availability layer?

Rollups need somewhere to post transaction data cheaply.
DA layers are purpose-built for exactly that — high-throughput, low-cost data publishing.

Without them, every L2 pays Ethereum L1 fees. That gets expensive fast.

2/ Celestia: Still the market leader

→ ~50% DA market share in 2026
→ First-mover + modular thesis proven out
→ Light nodes with DAS — verify data without downloading everything
→ Purpose-built blockchain for DA, nothing else

The OG modular chain. Others are catching up.

3/ EigenDA V2: The challenger

→ 100 MB/s throughput via DAC model
→ Plugs into the EigenLayer restaking ecosystem
→ Used by Mantle, Celo, and others already
→ Different trust model: relies on restakers, not independent consensus

Tradeoff: raw throughput vs decentralization assumptions.

4/ Avail: The technical dark horse

→ DAS + KZG commitments = stronger cryptographic guarantees
→ ~40 second finality
→ Polygon-incubated, now independent
→ Targeting both Ethereum rollups and new sovereign chains

The most technically rigorous option — but least battle-tested.

5/ Which should your rollup use?

Celestia → decentralization-first, proven track record
EigenDA  → EigenLayer ecosystem, enterprise-grade throughput
Avail    → KZG security guarantees, building something new

Full comparison + decision framework:
degen0x.com/learn/data-availability-layer-guide-2026
```
