## Tweet #336 — Flash Loans Explained
**Date:** Tue Apr 8, 11AM ET
**Type:** Educational thread (4 tweets)
**Topic:** Flash loans — the DeFi primitive most people misunderstand

### Tweet 1/4
```
Flash loans: borrow millions with zero collateral, repay in the same transaction.

Sounds like a hack. It's actually one of DeFi's most elegant primitives.

Here's how they actually work 🧵
```

### Tweet 2/4
```
The rule is simple: borrow any amount, but repay it + fees within the same block.

If the repayment fails → the entire transaction reverts. Like it never happened.

Zero risk to the lender. Zero collateral from the borrower.
```

### Tweet 3/4
```
Real use cases (not just exploits):

→ Arbitrage across DEXs
→ Collateral swaps on Aave/Compound
→ Self-liquidation to avoid penalties
→ Governance voting with borrowed tokens

The tool is neutral. The user decides.
```
